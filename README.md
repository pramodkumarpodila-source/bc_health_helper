# BC Health Helper

An AI-powered healthcare helper for BC patients. Provides care pathways, BC surgical wait time predictions, and lab result explanations.

## Features
- Symptom Helper with care pathway suggestions
- BC Wait Time Predictor (Prophet/LSTM)
- Lab Requisition Reader (OCR + NLP)
- Patient Clustering (K-Means)
- Streamlit Dashboard

## Tech Stack
Python, Pandas, Scikit-learn, Streamlit, Plotly, spaCy, HuggingFace, Prophet

## Disclaimer
Information only. Not medical advice. Call 911 in emergencies.
